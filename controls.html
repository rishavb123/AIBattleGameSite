Controls

Use the arrow keys to move

Use space bar to shoot

Special Keys:

R: Reset

C: Clear Score and Genes of Enemies

X: Pause

T: Train a Neural Network on this Session's Data (See Logs in Console)

S: Save the Model (See Logs in Console)

L: Load the Model (See Logs in Console)

P: Use the Model to Play

B: Bind a Key to a Saved Model

Esc: Clear Key Bindings
